# venture_pulse_ai# 📈 VenturePulse AI — Venture Capital Agent

VenturePulse AI is a high-performance web application that simulates a ruthless Indian Venture Capital (VC) strategist. Built entirely in Python, the platform leverages the advanced **Google Gemini 2.5 Flash** model to analyze raw startup or app concepts and provide direct, capital-efficient market viability critiques.

---

## 🚀 Core Features

- **Brutal Market Feasibility Analysis:** Evaluates business logic through the lens of a strict, real-world venture capitalist.
- **Indian Market Constraints:** Analyzes ideas against localized scaling challenges and legal/regulatory hurdles in India.
- **Sleek Web Interface:** Built with a clean, fully responsive GUI using Streamlit.
- **Secure Architecture:** Built with proper separation of concerns—decoupling API keys using cloud production environment secrets.

---

## 🛠️ Tech Stack & Architecture

- **Backend Logic:** Python 3.13+, Object-Oriented Programming (OOP)
- **AI Core Orchestration:** Google GenAI SDK (`gemini-2.5-flash`)
- **Frontend Presentation Layer:** Streamlit (Reactive UI Framework)
- **Version Control & Deployment:** Git, GitHub, Streamlit Community Cloud
